# data/

## `raw/`
Synthetic CSVs generated by `scripts/generate_synthetic_data.py` (patients,
comorbidities, prescriptions). Not versioned in git (see `.gitignore`) — each
developer generates their own local copy with a fixed seed for reproducibility.

## `processed/`
Derived datasets (local feature store): `training_set.parquet`,
`risk_model.joblib`. Generated by `scripts/build_training_set.py` and
`seniorrx.infrastructure.ml.train`.

## Data policy (LGPD/GDPR)

- **No real patient data must be placed in this folder.**
- If this project is adapted for real data, it must be anonymized beforehand
  (no name, national ID/SSN, address, phone, or medical-record number) and access
  must be restricted by role-based access control (RBAC), which is outside the
  scope of this repository.
- For versioning larger/real datasets, use DVC pointing to encrypted,
  access-controlled storage (see `configs/mlops.md`), never plain git.
